# ssl-main-lab

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面资源的集中存放与版本管理之用。  
所有页面内容均为独立编写，彼此之间无依赖关系。

## 目录说明

```
ssl-main-lab/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立的归档页面，文件名即页面标识。
- `assets/`：按需存放页面引用的资源文件，建议按页面名称分组。

## 页面归档说明

每个 HTML 页面均为自包含的完整文件，可直接在浏览器中打开运行。  
页面的内容、样式及功能均由自身实现，不依赖外部服务或数据库。  
归档页面的主要用途包括但不限于：

- 技术演示与功能测试
- 信息展示与资料整理
- 个人或团队的项目存档

如需新增页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保其完整性。
